Grounding Mechanism
Every sentence in the generated spec is checked against the artifact slice to ensure it has evidence:
UUID Reference Check
Sentence cites a UUID from the artifact slice → grounded
COBOL Identifier Check
Sentence mentions a data item / paragraph name from artifacts → grounded
Ungrounded Flagging
Sentences with no artifact reference are flagged for manual review
Grounding Score
grounded_sentences / total_sentences — shown in spec output header
