Grounding Mechanism
Every LLM sentence is checked against the artifact slice:
UUID Reference Check
Sentence mentions a UUID from the slice → grounded

Sentence mentions a data item / paragraph name from artifacts → grounded

Sentences with no artifact reference are flagged for manual review
